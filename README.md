# Week 3 ASSIGNMENT
The components only utilze `useState` hook from react and tailwind css
To Run:
 - clone the repo
  ```bash
    git clone [this repo usrl]
  ```
 - Ensure you have [nodejs](https://nodejs.org). Install the necessary deps using npm:
  ```bash
     npm install
  ```
  - Run the project:
  ```bash
      npm run dev
   ```
